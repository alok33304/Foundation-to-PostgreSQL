# Foundation-to-PostgreSQL
Overview
This repository is dedicated to practicing PostgreSQL for skill development. It focuses on strengthening SQL fundamentals, understanding relational database concepts, and improving hands-on database management skills.

Purpose
Build practical SQL skills

Practice schema design and query writing

Improve understanding of joins, constraints, and indexing

Enhance data handling and optimization techniques

Topics Covered
Database Design • CRUD Operations • Joins • Constraints • Indexing • Aggregations • Subqueries • Views • Transactions • Normalization

Tools
PostgreSQL • SQL
